*SET Items лепятся из вещей, которые указаны в описании "Комплектов". Если такой вещи нет в составе комплекта, то ничего не произойдёт.
RARE: Weapon / Armor / Ring / Amulet + 6 жемчужин = 1 SET Item ( Требуемый уровень к полученной вещи +5 )
RARE: Weapon / Armor / Ring / Amulet / Jewel / Arrows / Bolts + 10 жемчужин = 1 UNIQUE Item ( Требуемый уровень к полученной вещи +5 )
