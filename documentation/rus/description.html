Описание
Игру можно продолжить героем из оригинала D2LoD.
Можно использовать магию любого героя с кольцами или амулетами.
Артефакты и комплекты можно разобрать и получить металл для улучшения.
Можно улучшать раритетные вещи, выковать артефакт или сварить эликсиры.
Поднят уровень сложности, все Боссы стали сильней и получили приспешников.
Изменены все наёмники и добавлено четыре новых, у всех появилась новая магия.
Работают все Runewords. Добавлены: Jewelwords, Gemwords, Glyphwords и Pearlwords.
Множество новых артефактов, комплектов и рецептов.
Добавлены различные игрушки для вашей коллекции.
Добавлены эликсиры, органы, и временные вещи.
Множество новых вещей с клёвой графикой.
Новые убер локации, боссы и артефакты.
На всех уровнях играет новая музыка.
Возможны отверстия во всех вещах.
Более тридцати новых монстров.



Основные изменения:
- Новые заставки и меню из Diablo III.
- Заменена вся музыка в игре.
- Добавлены животные на все уровни.
- Новые текстуры травы первого акта и пальм второго акта.
 - Увеличено количество отверстий: Торс = 6, Шлем = 4.
- Теперь возможны отверстия и в других вещах: Перчатки = 2, Пояс = 2, Сапоги = 2, Кольца = 1, Амулеты = 1, Бросковое оружие = 3, Колчан = 1.
- Теперь шанс выиграть вещь равен ( 10% rare, 5% set, 1% unique ).
- Продаются Rejuvenation Potion.
- Продаётся всё оружие и броня.
- Можно выиграть: Руны, всё оружие и броню.
- Изменены цены.
- Изменён размер мешка ( 10x6 ), бесконечный сундук ( 10x10 ), вместилище куба Хорадримов ( 10x8 ), размер куба Хорадримов ( 1x1 ).
- 679 новых магических комбинаций ( Можно встретить магию, которая работает только днём или только ночью).
- Всем магическим вещам теперь соответствуют определённые цвета от магии ( мана на вещи даст голубой цвет, яд - зелёный, молния - жёлтый и т.д. ).
- Все названия зелий выделены определённым цветом.
- Свыше 170 новых картинок для различных вещей.
- Множество новых рецептов для ковки ( Crafted ).
- Теперь можно выковать: Джевелы, Чармы, Стрелы и Болты.
- Добавлено более 500 новых рецептов для куба + работают старые.
- Добавлено 16 Глифов, 8 Жемчужин и 1 новый камень, изменена руна ZOD.
- 90 Runewords ( работают все Runewords +9 новых ), 19 Jewelwords, 24 Gemwords, 60 Glyphwords и 12 Pearlwords ( цвет вещи изменится от камней, глифов и жемчужин ).
- Цвет названий всех руневордов изменён на "Пурпурный".
- 640 уникальных вещей ( 254 новых ) и 59 составных комплектов ( 20 новых для любого героя и по 1 для каждого ).
- Возможны стрелы и болты: магические, раритетные и уникальные.
- Возможны раритетные Чармы, новые уникальные Чармы и новые уникальные Джевелы.
- Убран лимит с уникальных вещей, кроме чарм и квестовых вещей.
- Теперь всё оружие и броня могут быть - ethereal.
- У Малаха в 5 акте вы сможете купить все три "убер ключа", цена = 3 млн. за каждый такой ключик.
- Добавлено 38 новых анимаций монстров, не заменяя старых.
- Создано 12 новых уникальных монстра.
- Переименованны уровни: "Swampy Pit" в "Resident Evil", "Drifter Cavern" в "Ice Prison".
- Создано 3 убер локации, 8 убер боссов, 8 статуэток ( чармы ), 12 убер артефактов и 3 убер комплекта. Названия убер локаций окрашены красным цветом.
- Максимальный уровень убер боссов - 120.
 - Заменён уровень "Forgotten Tower" на более красочный.
- Различные подсветки и раскраски уникальных монстров, боссов и убер локаций.
Сложность:
- Сопротивления на уровне сложности: Нормальная / Кошмар / Ад ( -25 / -100 / -250 ).
- Снятие опыта за смерть изменён: Нормальная / Кошмар / Ад ( -5% / -10% / -15% ).
- Опыт даётся за убийство монстра на 25% меньше.
- Статическая Область на уровне сложности Нормальная / Кошмар / Ад ( 20%, 33%, 50% ).
- Количество монстров изменено: Уникальные / Чемпионы ( мин-макс.  количество на сложности: Нормальная / Кошмар / Ад + 0-1 / 1-2 / 2-3 ).
- Количество монстров на всех уровнях увеличивается с уровнем игрока ( *1,25 - 2 ).
- Уровень монстров повышен на 5-10 уровней для сложности "Кошмар" и "Ад" соответственно.
- Добавлены случайные иммунитеты монстрам, на сложностях N/NM/H - 1 / 2 / 1.
- Максимальное количество иммунитетов на аду у простых монстров не больше 2 ( 1 - постоянный и 1 - случайный, либо 2 - постоянных ).
- Жизнь, защита и урон для всех монстров: мин-макс A1,A2,S1,El1-3  +20 / 40 / 60 ( Нормальная / Кошмар / Ад ).
- Шанс на критический удар для всех Боссов и для Супер уникальных теперь равен 10%. У убер боссов шанс на критический удар равен 15%.
- Статус для всех монстров увеличен на 25% на всех уровнях сложности.

Множество монстров на уровнях:
- Акт I : Cathedral, Tristram.
- Акт II : The Halls of the Dead Level 1-2-3.
- Акт III : Durance of Hate Level 3.
- Акт IV : Chaos Sanctum.
- Акт V : Hell 1-2-3, Throne of Destruction.
- На всех Убер уровнях.

Уникальные монстры и боссы:
*У актовых боссов жизни повышены на 100%. У убер боссов жизни повышены на 50%.
*Защита и урон у актовых и убер боссов повышены на: 30 / 60 / 90 ( Нормальная / Кошмар / Ад )
*Все актовые и убер боссы получают сопротивления на уровнях сложности Нормальная / Кошмар / Ад ( 25 / 50 / 75 ).
*Уровень АУРЫ на уровнях сложности Нормальная / Кошмар / Ад ( 5 / 10 / 20 ).
*У всех супер уникальных боссов защита и урон повышены на: 20 / 30 / 40 ( Нормальная / Кошмар / Ад ).

Эликсиры:
- Маленький эликсир: +1 к силе, +1 к ловкости, +1 к живучести ( макс жизни +3, стамина +1 ) , +1 к энергии ( макс мана +2 ).
- Большой эликсир: Demon's Red Blood ( +5 очков статуса ).
- Большой эликсир: Demon's Blue Blood ( +1 навык для распределения ).

Вещи на время (используются как зелья, время действия 5 МИНУТ):
- Torch: даёт бонус: +250% найти магическую вещь, +500% к выпадению золота из монстра, +15 к освещению.
- Hero's Cigarette: бег +100%, регенерация маны +200%, регенерация жизни +200%.
- Super Elixir: защита, жизнь, мана +100%.
- The Holy Drink: ко всем навыкам +3.
- Monster's Hand: быстрая атака и произнесения заклинания +200% , к атаке +100%.
- Monster's Skull: поднимает скелетов из мёртвых монстров, урон по демонам и трупам +100%.

Разное:
- 16 Glyphs: новые руны, но действуют, как камни ( нет названий глифов при вставке в вещь, меняют цвет вещи ).
- Органы: ингредиенты для улучшений вещей.
- FLAG: шаманская вещь, необходимая для крафта.
- Black Oil: повышает защиту на любой броне +1, или +1 к максимальному физическому урону для любого оружия.
- Destroyer Malus: необходимый молоток для разрушения различных вещей ( продаётся у всех кузнецов ).
- Металлы: Adamantium, Rubidium, Titanium, Chromium. С их помощью можно улучшить раритетные вещи: оружие, всю броню, щиты, кольца и амулеты.
- 8 Pearls: с их помощью можно создать артефакт или часть комплекта, получить жемчуг можно из одного совершенного камня и 10 металлов или выбить из убер боссов.
- 67 Игрушек: можно разбить или улучшить различные вещи.
- Platinum: с помощью этой руды можно хорошо улучшить шмотки, которые будут давать значительные свойства в разное время суток. Также можно разбить и получить неплохую шмотку.
- Silver: разрушив эту руду, вы получите полный комплект вещей для героя.

- Убер боссы ( NM ): любой орган 1к17 / флаг 1к70 / чёрное масло 1к70 / вещь на время 1к70 / игрушка 1к37 / металл 1к37 / платина 1к300 / жемчуг 1к75 / большой эликсир 1к300
- Убер боссы ( H ): любой орган 1к12 / флаг 1к50 / чёрное масло 1к50 / вещь на время 1к50 / игрушка 1к25 / металл 1к25 / платина 1к200 / жемчуг 1к50 / большой эликсир 1к200
